# Arquitectura — DentApp Landing Page

Documentación de arquitectura usando el modelo [C4](https://c4model.com/):
diagramas jerárquicos que van de lo general (contexto) a lo específico
(contenedores, componentes, código), cada uno como "zoom" del anterior.

Para este proyecto solo se documentan los primeros dos niveles — un sitio
estático de una sola landing no justifica bajar a Nivel 3 (componentes) o
Nivel 4 (código).

## Índice

| Nivel | Diagrama | Qué muestra |
|-------|----------|-------------|
| 1 | [Contexto](c4-nivel1-contexto.md) | El sistema como caja negra: quién lo usa y con qué interactúa |
| 2 | [Contenedores](c4-nivel2-contenedores.md) | Las piezas técnicas internas del sistema (páginas, estilos, scripts) |

Ver también: [docs/adr/](../adr/) para las decisiones de arquitectura asociadas.
